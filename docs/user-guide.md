## 快速开始

### 首次启动

```bash
cp .env.example .env   # 创建环境变量文件
npm install            # 安装依赖
npm start              # 启动服务器
```

访问 `http://localhost:3000`，默认管理员账号:
- 用户名: `admin`
- 密码: `admin123`

**安全提示**: 首次登录后请立即修改默认密码！

### 管理员注册

普通用户可直接注册，管理员注册需提供管理员密钥（在 `.env` 文件中设置 `ADMIN_SECRET`）。

### 密码要求

- 长度: 6-128 个字符
- 必须包含: 至少一个字母和一个数字
- 建议: 使用大小写字母、数字和特殊字符的组合

## 用户系统

### 注册与登录

1. 点击导航栏的「登录」按钮
2. 切换到「注册」标签填写用户名和密码
3. 如需注册管理员账号，勾选「管理员」，并填写管理员密钥

### 个人资料

登录后点击导航栏用户头像或用户名，选择「我的主页」：

- **编辑资料**: 点击「编辑个人信息」，可修改昵称、简介、头像、技能标签和社交链接
- **修改密码**: 在编辑页面底部输入当前密码和新密码
- **他人视角**: 点击「他人视角」查看自己的公开主页

### 头像

支持上传头像，上传时可裁剪为 1:1 方形。支持 jpg/png/gif/webp 格式。

## 帖子系统

### 帖子分类

| 分区 | 权限 | 用途 |
|------|------|------|
| 作品区 (Work) | 仅管理员发布 | 展示正式作品 |
| 聊天区 (Chat) | 所有用户可发布 | 日常交流讨论 |

### 发布帖子

1. 点击导航栏的「发布作品」或「发帖」
2. 填写标题、简介、标签（逗号分隔）
3. 可选封面图片（支持裁剪）
4. 添加内容块: 文本、图片、视频、代码

#### 内容块

每个帖子由多个内容块组成，支持排序（↑↓按钮）和删除：

- **文本块**: 多行文本
- **图片块**: 上传图片或输入图片 URL
- **视频块**: 上传视频或输入视频 URL
- **代码块**: 代码编辑器，自动语法高亮

### 帖子操作

- **预览控制**: 编辑时可为内容块设置「允许普通用户预览」
- **置顶/精华**: 管理员可将帖子置顶或标记为精华
- **锁定**: 管理员锁定帖子后，用户无法添加新评论

### 帖子互动

- **点赞/点踩**: 点击帖子底部的 👍/👎 按钮
- **收藏**: 点击 📖 按钮将帖子收藏到收藏夹
- **浏览计数**: 系统自动统计浏览次数（同一用户仅计一次）

## 评论系统

### 发表评论

- 在帖子底部输入框填写内容，点击「发布评论」
- 支持 Ctrl+Enter 快捷发布

### 回复评论

- 点击评论下方的「回复」按钮
- 回复会自动嵌套在原评论下方

### 编辑/删除评论

- 自己的评论可编辑或删除
- 管理员可编辑/删除任何评论
- 删除评论会同时删除其所有子评论

### 评论通知

- 当有人评论你的帖子或回复你的评论时，你会收到通知
- 点击通知会自动跳转到帖子详情并滚动到对应评论位置
- 评论区会高亮显示目标评论

## 好友系统

### 添加好友

1. 访问用户的主页
2. 点击「添加好友」按钮发送请求
3. 对方收到通知后，可在好友页面接受或拒绝

### 好友管理

- 在「好友」页面查看好友列表和在线状态
- 可移除好友
- 好友请求数量会显示在导航栏

### 在线状态

- 在线检测: 5 分钟内活跃的用户显示为在线（绿色圆点）
- 好友列表每 30 秒自动刷新在线状态

## 私信

1. 在好友列表中点击「💬 私信」按钮
2. 进入聊天页面发送消息
3. 未读消息数量显示在导航栏

## 音乐播放器

### 基本功能

底部固定音乐播放器，支持:
- 播放/暂停 (空格键)
- 上一首/下一首
- 进度拖动
- 音量调节
- 播放模式切换: 顺序播放、循环播放、单曲循环

### 音乐管理（管理员）

1. 进入「音乐」页面
2. 上传歌曲（名称、艺术家、封面）
3. 编辑歌曲信息
4. 创建和管理播放列表

### 公开播放列表

- 播放列表可设为公开，其他用户可查看和收藏
- 公开播放列表有独立的收藏计数

### 键盘快捷键

| 键 | 功能 |
|----|------|
| 空格 | 播放/暂停 |
| → | 下一首 |
| ← | 上一首 |
| M | 切换静音 |

## 收藏夹

### 管理收藏夹

1. 进入「收藏」页面
2. 创建新的收藏夹（输入名称点击创建）
3. 点击收藏夹名称切换查看其中的收藏

### 收藏帖子

- 在帖子详情页点击 📖 按钮
- 选择要收藏到的收藏夹
- 已收藏的帖子会在收藏夹中显示

### 管理收藏

- **单个删除**: 点击收藏项右侧的 ✕ 按钮
- **批量删除**: 勾选多个收藏项，点击「删除选中」按钮
- **全选**: 点击「全选」复选框选择所有收藏

## 举报系统

### 提交举报

在以下位置可提交举报:
- **帖子详情页**: 点击 🚩 按钮
- **用户主页**: 点击「举报用户」按钮

举报需要填写原因，管理员会收到待处理通知。

### 管理举报（管理员）

管理员可在管理面板的「举报管理」中:
- 查看待处理、已处理、已驳回的举报
- **处理**: 标记为已处理（完成审查）
- **驳回**: 标记为无效举报
- **禁言**: 若被举报者为用户，可直接禁言（支持限时/永久）
- **锁定/删除**: 若被举报者为帖子，可直接锁定或删除

## 等级与权限

### 获取经验值 (XP)

| 操作 | XP |
|------|----|
| 发帖 | +20 |
| 评论 | +10 |
| 收到点赞 | +2 |
| 帖子加精 | +50 |
| 收到回复 | +5 |

### 等级分区

不同等级可访问不同分区:
- **作品区**: 基础等级即可访问
- **聊天区**: 较高等级才能访问
- **音乐区**: 特定等级可访问

管理员不受等级限制。

### 查看等级

- 导航栏显示当前等级徽章和 XP
- 可在「我的主页」查看详细等级信息
- 管理面板可配置等级名称、图标和阈值

---

## 管理面板

仅管理员可访问。管理面板包含以下功能:

### 数据概览

显示分区统计（帖子数、评论数）、用户总数、待处理举报数。

### 举报管理

见 [举报系统](#举报系统) 章节。

### 用户管理

| 功能 | 说明 |
|------|------|
| 用户列表 | 分页显示所有用户，支持搜索 |
| 禁言 | 临时禁言（指定小时数）或永久禁言 |
| 解禁 | 解除禁言状态 |

禁言后的用户:
- 无法发布帖子或评论
- 无法发送好友请求或私信
- 无法点赞、收藏、举报
- 无法上传音乐或修改个人资料
- 登录时显示禁言横幅（含剩余时间）
- 收到禁言通知

### 等级管理

- 配置等级名称、图标、背景图
- 设置各等级所需 XP
- 设置各等级可访问的分区
- 手动调整用户等级和 XP

### 批量操作

- 批量删除帖子
- 批量删除收藏
- 批量删除音乐/播放列表

## 主题与设置

### 亮色/暗色主题

点击导航栏的 🌙/☀️ 按钮切换主题。偏好设置存储在浏览器 localStorage 中。

### 声音设置

- 调整个性化提示音的音量
- 管理员可上传自定义点击音效（MP3 格式，最大 500KB）

### 导航

- 桌面端: 顶部导航栏
- 移动端: 汉堡菜单，侧边抽屉导航
- 侧边菜单显示分区访问状态（锁定/解锁）

## 快照与回滚（开发者）

### 创建快照

修改代码前创建快照以保存当前状态:

```bash
bash scripts/snapshot.sh "修改前的状态"
```

### 回滚

```bash
# 查看所有快照
bash scripts/rollback.sh list

# 回滚到指定版本
bash scripts/rollback.sh <commit-hash>
```

支持硬回滚、软回滚和单文件恢复三种模式。
