# 更新日志

## 2026-05-16

### 新增
- **个人主页仪表盘**: `#/profile` 从简单资料卡升级为仪表盘，包含统计面板（作品数、评论数、获赞数、注册天数）、最近作品网格、最近通知列表
- **侧边菜单入口**: 新增「👤 我的主页」菜单项，位于侧边菜单首位
- **用户统计 API**: `GET /api/users/:id/stats` 返回聚合统计数据
- **他人视角切换**: 个人主页可切换到他人视角查看自己的公开页面
- **返回按钮**: 他人视角页面新增「← 返回我的主页」按钮
- **我的作品页面**: `#/my-posts` 路由，展示当前用户所有已发布作品，支持分页加载，每张卡片可点击跳转详情，顶部有返回个人主页按钮

### 修复
- **路由竞态**: `renderMyProfile` 拆分为同步骨架 + 异步数据加载（`setTimeout` 延迟），避免 `_handlePath` 中 `updateNav` 等同步操作与异步渲染的时序冲突
- **他人视角返回失效**: 他人视角按钮从直接调用 `renderUserProfile()` 改为 `Router.navigate('#/users/{id}')`，使 URL hash 真正变化，返回按钮的 hashchange 事件得以触发

### 安全
- **CSRF 保护**: Origin/Referer 头验证
- **限流**: 认证接口 10 次/分钟
- **文件上传**: 禁止 .html/.js 上传
