🔴 当前问题

  • 所有字段纵向堆叠，视觉无分区
  • 封面上传区域简陋（仅文字 + 按钮）
  • 内容块 bar 无背景色区分
  • 块操作按钮（↑↓✕）为纯文本
  • 标签输入需手动点击「添加」
  • 附件等级/积分设置分散
  • 无分类选择（聊天区无法切换）
  • 编辑器整体缺少呼吸感

🟢 A · 表单优化

  • 分区标题（基本信息 / 封面 / 内容块）
  • 封面区域改为预览图 + 文件信息 + 操作
  • 块头部有背景色 + 类型徽章
  • 操作按钮改为图标按钮组
  • 标签输入框 + 添加按钮 同行
  • 双列网格（分类 + 标签同行）
  • 统一下边距和间距

🟢 B · 双栏编辑布局

  • 左侧 1fr：标题 + 简介 + 内容块 + 操作
  • 右侧 280px sticky 侧边栏
  • 侧边栏：封面 → 标签 → 分类 → 高级设置
  • 内容区不再被元信息打断
  • 编辑体验更接近 Notion / WordPress
