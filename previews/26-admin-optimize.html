📐 三方案对比

A · 标签卡片化

  • 卡片网格布局，每标签独立卡片
  • 实时搜索过滤
  • 分类切换（全部/作品/聊天）
  • hover 露出操作菜单
  • 虚线添加输入框

B · 分组 + 热度

  • 列表视图 + 热度颜色条
  • sticky 侧边栏统计数据
  • hover 露出编辑/删除按钮
  • 标签合并功能（批量选中后合并）
  • 热度分布图例

C · 举报卡片化

  • 顶部统计卡片（总/待处理/已处理/驳回）
  • 可展开/收起详情面板
  • 状态指示圆点（黄色/绿色/灰色）
  • 内联操作按钮组
  • 搜索 + 状态标签页筛选
