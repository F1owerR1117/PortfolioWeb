📐 三种方案对比

A · 沉浸式阅读

  • 窄栏居中（740px 最优阅读宽度）
  • sticky 粘性反应栏
  • 大标题 32px · 宽幅封面
  • 底部相关推荐
  • 适合长文深度阅读

B · 双栏布局

  • 内容 + 300px 侧边栏
  • 侧边栏：作者卡 + 反应 + 目录 + 标签
  • 充分利用宽屏空间
  • 粘性侧边栏（滚动跟随）
  • 适合信息型 / 教程类帖子

C · 杂志式

  • 380px Hero 封面图 + 标题叠加
  • 首字下沉 + 引文区块
  • 作者栏在封面下方
  • 评论气泡风格
  • 适合创意 / 作品展示类帖子
